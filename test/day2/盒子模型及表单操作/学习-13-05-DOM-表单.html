获取表单和表单元素；

  1. document.forms：返回页面中所有表单元素；
  2. 表单和表单元素可以用标准的节点获取方式来获取，不多说了，瞅两个小例子； 
                        //获取id是myform表单下所有的复选按钮
                    document.querySelector('#myform input[type="checkbox"]');
                    //获取id是myform表单下所有的name是newsid的复选按钮
                    document.querySelector('#myform input[type="checkbox"][name="newsid"]');
  3. 表单和表单元素可以用标准的节点获取方式来获取，不多说了；

表单和元素的属性和方法；

  1. 表单.submit()：表单提交；
    表单.reset()：表单重置；
  2. 元素.type：只读，表单元素类型； 
                        var type = document.querySelector('#myform input[name="email"]').type;
                    console.log(type);
    元素.form：只读，包含该元素的form表单对象，不存在返回null； 
                        var form = document.querySelector('#myform input[name="email"]').form;
                    console.log(form);
    元素.name：只读，元素的名称； 
                        var name = document.querySelector('#myform input[name="email"]').name;
                    console.log(name);
    元素.value：可读可写，简单理解就是输入框的值； 
                        var email   = document.querySelector('#myform input[name="email"]');
                    email.value = '1410475107@qq.com';
                    var value   = email.value;
                    console.log(value);

表单和元素的事件处理程序；

  1. 表单.onsubmit：监听表单提交，表单提交到服务器前，先执行该事件；
    如果事件处理程序返回false，则表单提交取消，数据不会提交到服务器处理；
    需要注意的是，表单.submit()方法不会触发onsubmit事件； 
                        document.querySelector('#myform').onsubmit = function () {
                        console.log('表单不提交，我要返回false');
                        return false;
                    }
    onreset，表单重置事件，处理过程和onsubmit一样，不说了不说了；
