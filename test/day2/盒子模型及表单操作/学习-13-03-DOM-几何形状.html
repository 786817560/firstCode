查询元素的几何尺寸；

  1. 节点.getBoundingClientRect()：得到一个元素的尺寸和坐标位置；
    left和top对应的是左上角x,y坐标；
    right和bottom对应的是右下角x,y坐标； 
                       //html代码
                    < div class="mydiv">
                        这是我的盒子
                    < /div>
                    //JS代码
                    var mydiv = document.querySelector('.mydiv');
                    var clent = mydiv.getBoundingClientRect();
                    console.log(clent);
    NOTE: 
                        //其中的width和height代表的是元素的宽度和高度
                    clent.width     = clent.right - clent.left;
                    clent.height    = clent.bottom - clent.top;
    所以，浏览器返回的ClientRect里面如果没有width和height， 可以通过上面的公式计算出来；
  2. 节点.scrollIntoView()：让元素在视口中可见，默认将元素和视口的上边缘对齐; 如果传递参数false，则将元素的下边缘和视口的下边缘对齐； 
                        //html代码：要控制的div
                    < div class="noview">开始你是看不到我的，我在下面隐藏着了< /div>
                    < div class="height">测试用< /div>
                    //HTML代码，绑定事件的按钮
                    < button class="btn btn-warning show">让我看到他< /button>
                    //JS代码
                    document.querySelector('.show').onclick = function () {
                        document.querySelector('.noview').scrollIntoView(false);
                    }
